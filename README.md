# מדברים פסח 🍷 (Pesach Conversation Engine)

### **Interactive Engagement Engine for Multi-Context Connectivity**
A lightweight, high-performance web application built with **Zero Dependencies** to facilitate meaningful interactions through the symbolism of the Passover Seder.

---

## 🚀 Overview
**מדברים פסח** is a dynamic "Icebreaker" engine designed to bridge gaps between team members and family members alike. Developed using **Google's Antigravity IDE**, this project transforms traditional holiday symbols into an interactive "Card-Flip" experience.

The project demonstrates that fluid, professional user experiences can be achieved using the core fundamentals of the web: **HTML5, CSS3, and Vanilla JavaScript.**

## 🛠️ Technical Stack
* **IDE:** Developed and optimized via **Google Antigravity**.
* **HTML5:** Semantic structure for optimal accessibility and clean DOM hierarchy.
* **CSS3:** Advanced Grid/Flexbox layouts, 3D transforms (`preserve-3d`), and native transitions for a premium "Native-app" feel.
* **Vanilla JavaScript (ES6+):**
    * **State Management:** Dual-mode state persistence via `localStorage`.
    * **Randomizer:** Smart randomization with anti-repeat logic.
    * **Async Data-Loading:** Concurrent fetching of multiple datasets.

## ✨ Key Features
* **Dual Game Modes:** Seamlessly toggle between:
    * **Team Mode (מצב צוות):** Professional and bonding-focused icebreakers for work environments.
    * **Family Mode (מצב משפחה):** Personal and reflective prompts for the Seder table.
* **Wavy Card Flip:** A premium 3D transition effect that flips the entire board when switching modes, staggered across the grid for a "wave" feel.
* **Dual-Depth Logic:** Every symbol offers two levels of engagement:
    * *Shallow (קליל):* Light, fun, and quick icebreakers.
    * *Deep (עמוק):* Reflective, professional, and personal growth prompts.
* **Smart Randomizer:** Integrated logic pulls from the active dataset and ensures you don't see the same question twice in a row.
* **Responsive Overlay System:** A focus-driven UI that brings content to the center of the screen, optimized for remote presentation or in-person display.

## 📖 How it Works
1.  **Select Your Mode:** Use the toggle at the top to choose between "Team" and "Family" contexts.
2.  **Select a Symbol:** Choose from 8 iconic Passover symbols representing different life/work themes.
3.  **Choose Your Depth:** Select "Shallow" for a quick laugh or "Deep" for a meaningful conversation.
4.  **Engage:** The app presents a randomly selected question. Need a change? Hit the 🎲 **Shuffle** button.
5.  **Rotate:** Return to the grid and move to the next participant.

## 🏗️ Architecture Philosophy
This project follows a "Back to Basics" philosophy. Developed within the **Antigravity** environment, it prioritizes **clean, maintainable, and dependency-free code**, proving that fundamental web technologies are the ultimate tool for rapid, reliable deployment.
