# Pesach 🍷

### **Interactive Engagement Engine for Team Connectivity**
A lightweight, high-performance web application built with **Zero Dependencies** to facilitate meaningful team interactions through the symbolism of the Passover Seder.

---

## 🚀 Overview
**Pesach** is a dynamic "Icebreaker" engine designed to bridge the gap between remote team members. Developed using **Google's Antigravity IDE**, this project transforms traditional holiday symbols into an interactive "Card-Flip" experience for high-engagement remote sessions (Microsoft Teams/Zoom).

The project demonstrates that fluid, professional user experiences can be achieved using the core fundamentals of the web: **HTML5, CSS3, and Vanilla JavaScript.**

## 🛠️ Technical Stack
* **IDE:** Developed and optimized via **Google Antigravity**.
* **HTML5:** Semantic structure for optimal accessibility and clean DOM hierarchy.
* **CSS3:** Advanced Grid/Flexbox layouts and native transitions for a "Native-app" feel without the overhead of external libraries.
* **Vanilla JavaScript (ES6+):** * **Custom State Management:** Precise control over category switching.
    * **Randomization Algorithms:** Ensures unique user experiences across 8 thematic categories.
    * **Direct DOM Manipulation:** Maximum performance with zero latency during screen-sharing.

## ✨ Key Features
* **Dual-Depth Logic:** Every symbol offers two levels of engagement:
    * *Shallow:* Light, fun, and quick icebreakers.
    * *Deep:* Reflective, professional, and personal growth prompts.
* **Smart Randomizer:** Integrated logic pulls random questions from an extensible data object.
* **Responsive Overlay System:** A focus-driven UI that brings content to the center of the screen, optimized for remote presentation.
* **Instant Reset:** Seamless navigation back to the "Command Center" (Main Grid) for fast-paced group sessions.

## 📖 How it Works
1.  **Select a Symbol:** Choose from 8 iconic Passover symbols representing different life/work themes.
2.  **Choose Your Depth:** Select "Shallow" for a quick laugh or "Deep" for a meaningful conversation.
3.  **Engage:** The app presents a randomly selected question. Need a change? Hit the 🎲 **Shuffle** button.
4.  **Rotate:** Return to the grid and move to the next participant.

## 🏗️ Architecture Philosophy
This project follows a "Back to Basics" philosophy. Developed within the **Antigravity** environment, it prioritizes **clean, maintainable, and dependency-free code**, proving that fundamental web technologies are the ultimate tool for rapid, reliable deployment.
